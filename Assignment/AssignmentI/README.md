這份作業（Assignment I）的主要目標是讓你熟悉 **GitHub** 的基本操作，並建立未來課程中用來記錄學習內容的空間。

以下是作業內容的重點整理：

### 1. 核心任務：建立兩個 Repository

你需要在 GitHub 上建立兩個存放空間（Repositories）：

* **課程紀錄庫**：
* 名稱必須為：`11401_CS203A` 。


* 用途：作為這門課的「活文件」，記錄你的學習心得與內容 。


* 要求：必須包含一個 `README.md` 檔案，內容要有課程資訊、你的姓名學號以及 Repo 的簡介 。




* **個人網頁庫**：
* 名稱必須為：`你的帳號名稱.github.io` 。


* 用途：開啟 **GitHub Pages** 功能後，這會成為你的個人課程首頁 。
